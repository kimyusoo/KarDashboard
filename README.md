# 🏠 KAR 부동산 인사이트 (MVP)

한국공인중개사협회 회원용 **주간·월간 부동산 분석 대시보드**.
한국부동산원 R-ONE OpenAPI(주간 아파트 가격동향)와 공공데이터포털 국토교통부 실거래가
API를 기반으로, 전국 시장 동향을 회원이 한눈에 보고 고객 상담에 활용하도록 구성했습니다.

## 현재 구현 (MVP)

| # | 화면 | 데이터 출처 | 상태 |
|---|------|------------|------|
| 1 | 이번 주 시장 동향 요약(자동 코멘트) | 부동산원 주간지수 | ✅ |
| 2 | 매매·전세 가격지수 + 추이 차트 | 부동산원 | ✅ |
| 3 | 시도별 변동률(매매/전세) 바차트 | 부동산원 | ✅ |
| 4 | 시군구 상승률 TOP10(매매/전세) | 부동산원 | ✅ |
| 5 | 전국 시군구 코로플레스 지도(매매/전세 토글) | 부동산원 + GeoJSON | ✅ |
| 6 | 월세 동향(월간 매매/전세/월세 지수 + 시도별) | 부동산원 월간 | ✅ |
| 7 | 단지별 평균 거래가 상승률 TOP(실거래) | 공공데이터 실거래 | ✅ |
| 8 | 아파트 거래량·평균가(샘플 6개 지역) | 공공데이터 실거래 | ✅ |
| + | 지도 시군구 클릭 → 담당 협회 지회 연결 | kar.or.kr 지회 | ✅ |
| + | 심화분석(ISCU) 연동 탭 `/insights` | 서울사이버대 5개 시스템 | ✅ |
| + | 전국 지회 안내·검색 `/branches` (19시도회 241지회) | kar.or.kr | ✅ |
| + | 회원 로그인(간이 인증) + 주간/월간 PDF 리포트 `/report` | 미들웨어·인쇄CSS | ✅ |

> 데이터 정확성 검증: 2026-06-01 기준 전국 매매 +0.07%, 전세 +0.11%, 서울 매매 +0.25%,
> 경기 동탄구 +0.60% 등 한국부동산원 공식 보도자료와 수치 일치 확인 완료.

### 코로플레스 지도 구현 메모
- GeoJSON: kostat 2018 시군구 250개 → mapshaper로 12% 단순화·clean(1.3MB)
- REB 지역명 매칭: `시도|시군구명` 키(일반구는 `시명+구명` 결합), 약 83% 매칭
- ⚠️ d3-geo 렌더링 시 **폴리곤 winding이 시계방향이면 전 지구로 해석**되어 지도가 깨짐
  → `@mapbox/geojson-rewind`(clockwise=true)로 CCW(구면) 정렬 필수 (geoArea≈0.0025 확인)

## 기술 스택

Next.js 15 (App Router) · React 19 · TypeScript · Recharts · d3-geo(코로플레스) · ISR 캐시(6시간)

## 로컬 실행

```bash
npm install
cp .env.local.example .env.local   # API 키 입력
npm run dev          # http://localhost:3000
```

## 환경 변수

| 키 | 설명 |
|----|------|
| `REB_API_KEY` | 한국부동산원 R-ONE OpenAPI 인증키 |
| `PUBLIC_DATA_KEY` | 공공데이터포털 실거래가 인증키(Decoding) |
| `SITE_PASSWORD` | 회원 접속 비밀번호(간이 인증) |
| `SITE_AUTH_TOKEN` | 로그인 쿠키 검증 토큰(임의 비밀문자열) |

## Vercel 배포

1. GitHub 레포에 푸시
2. Vercel에서 Import → 환경변수 2개 등록 → Deploy
3. `vercel.json`의 Cron(화·목 06:00 KST 부근)이 페이지 재생성을 트리거
